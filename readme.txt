== Description ==

This is a simple plugin, targeted for responsive themes, that adds a resize me graphic to the bottom right corner of the website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `wp-resize-me` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where are the settings. =

There are no settings for this plugin. Just activate it and it will work. Deactivate it to turn it off.
